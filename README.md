# Grade Management System

## Overview
This project is a Java-based Grade Management System that stores student marks, calculates averages, assigns grades, and generates performance reports.

## Features
- Add student marks
- View student performance
- Calculate averages
- Assign grades (A–F)
- Identify top performer
- Performance statistics

## Technologies
- Java
- Arrays
- Scanner (input)

## How to Run
1. Open terminal in src folder
2. Compile:
   javac GradeManagementSystem.java
3. Run:
   java GradeManagementSystem
